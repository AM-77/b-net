# b-net

  A very basic & simple python botnet.

## dependencies installation

  run `` pip install -r requirements.txt ``
  
## execute

  add execution mode: `` chmod +x b_net.py ``
  <br>
  run: `` ./b_net.py ``
  <br>
  or
  <br>
  just run: `` python b_net.py ``
